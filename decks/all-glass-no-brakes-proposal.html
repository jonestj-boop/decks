What We're Building From Scratch

Your mockup covers the design — but a live store needs more. Here's what we add.

Item Why It's Needed
Cart page Not in mockup — built in AGNB brand style with AJAX updates
Full product page Variant selects, image gallery, quantity, Add to Cart button wired to Shopify
Commission form handling Form submissions routed to email + Klaviyo or Shopify inbox
Shopify navigation Nav menus wired to Shopify admin so client can edit links without code
Collection → Liquid loop Static collection cards replaced with live Shopify collections data
