C-02
Add Google Reviews Widget Sitewide
Low Effort
What to do
  1. Go to elfsight.com, create a free account, and add a new Google Reviews widget.
  2. Search for "Party Kids America" or paste your Google Business Profile URL. Configure: 5-star filter on, carousel layout, show reviewer name + photo.
  3. Copy your Elfsight App ID from the dashboard (looks like elfsight-app-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx).
  4. In ERS, go to /cp/source_code/ and paste the sitewide script just before </body>.
  5. To show reviews on a specific page (e.g. homepage), go to /cp/website_pages/ → open the page → add a Custom Code block → paste the widget div tag.
Deliverable — Embed Code
💡Paste the script tag in /cp/source_code/ (sitewide), and place the div tag wherever you want reviews to appear on a page.
