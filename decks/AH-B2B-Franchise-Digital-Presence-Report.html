02

Two audiences, one domain

The keywords a furniture retailer chases and the keywords a franchise or trade program needs are not the same thing, and treating them as one search strategy is the core issue on adrianahoyos.com today. Consumer-style discovery terms still matter — interior designers behave as prosumers, searching product terms like "sofa bed" or "bedroom ideas" for creative reference long before they ever look for a trade program. Those terms earn a real place in the strategy. But they sit above a second, currently empty layer: people typing "furniture franchise opportunities," "furniture wholesalers," or "interior design trade program" — searches the .com does not appear for at all.

Adriana Hoyos needs both layers working at once: the consumer/prosumer discovery layer it already has some presence in, and the B2B/franchise layer that has to be built from the ground up. Every recommendation in this report is organized around keeping the first and originating the second.

.com / .us site architecture

The domain split below is the organizing structure for the US rollout, built specifically around the B2B/franchise goal:

DomainJobStructure
.com Global brand narrative + the B2B/franchise front door: trade program, dealer inquiries, "open a showroom" pathway. One canonical H1 and one target keyword per page; a real Franchise/Trade hub replacing the current bare login walls.
.us Localized SEO and campaign infrastructure for the US rollout, built to scale to each future franchisee. City-based subfolders from day one — /us/dc, /us/austin, /us/coral-gables — paired one-to-one with dedicated Meta ad accounts per market.
Design Studio subdomain A separate design-services offering, unrelated to the furnishing business and out of scope for this analysis. Keep unindexed via robots.txt so it stops competing with the furnishing business for crawl and keyword signal.

07

Phased recommendations

Sequenced critical-first, structural-second, scale-ready-last, so SEO and CRO work can run as one coordinated plan.

01Foundation

Fix the architecture that's already broken

Consolidate the homepage to one H1; resolve the 20+ page conflict around "Design Studio"; noindex the design-services subdomain; request do-follow links on the next Architectural Digest and Robb Report placements specifically (Elle Decor already links do-follow).

02Build the front door

Stand up the .com B2B hub and launch .us

Publish a real Trade Program / Become a Franchisee page — staged like Caracole and Bernhardt, with context before the login wall, not a bare gate. Launch .us with city-based subfolders and individual location pages, starting with the DC and Coral Gables showrooms.

03Scale-ready

Template the franchisee playbook

Turn the .us city-page structure into a repeatable template — local SEO, local schema, and a matching Meta account — so each new franchisee after Sean launches on the same infrastructure instead of improvising.
